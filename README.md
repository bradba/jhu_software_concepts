# Portfolio Flask App

Simple Flask portfolio scaffold with a homepage (bio + image) and a Projects page.

Repository: https://github.com/bradba/jhu_software_concepts

Quick start (zsh):

1. Create and activate a virtual environment

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install requirements

```bash
pip install -r requirements.txt
```

# Portfolio Flask App

Simple Flask portfolio app for coursework.

Repository: https://github.com/bradba/jhu_software_concepts

Quick start (zsh):

1. Create and activate a virtual environment

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install requirements

```bash
pip install -r requirements.txt
```

3. Run the app (defaults to localhost:8080)

```bash
python app.py
```

Open `http://127.0.0.1:8080` in your browser.
